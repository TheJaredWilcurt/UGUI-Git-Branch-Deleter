System Requirements:

  • 85MB of avaliable disk space
  • 55MB of avaliable memory
  • Windows Vista, 7, 8, or 10 (XP may work but is not officially supported). Currently only supplying a 32-Bit version of UGUI: Git Branch Deleter for this platform. Will create a 64-bit if requested.
  • OSX 10.7 and above (older versions are not officially supported and are not likely to work). Currently we are not pre-packaging for this OS, so you'll need NPM installed to do use it.
  • Ubuntu 12 and 14 (Ubuntu 10 should work, other versions of Linux, especially Debian based, may work but only Ubuntu 12 and 14 are officially supported). Currently we are not pre-packaging for this OS, so you'll need NPM installed to do use it.
  • Only x86 and x64 architectures are supported (32-Bit or 64-Bit)

Installation instructions:

  • Windows: Download, Unzip, Run exe.
  • Non-Windows: Download, Unzip, npm install, npm start

Features:

  • Point to any local git repo to see a list of all branches
  • Select any branch, then click the Delete! button (runs git branch -D BRANCHNAME)
  • Type in the URL for your GitHub repo to have a clickable link to your online branch
  • Type in the URL to your ticket management system (includes Clubhouse, Jira, and GUS)
  • Links to your associated tickets will be available to verify the ticket is closed before deleting the branch
  • With Clubhouse you can add your API token and it will also display how many days since each branch's story was closed
  • Clubhouse stories that were completed 30+ days ago will be bold
