About UGUI: Git Branch Deleter

This application allows you to select a Git repository, list all local branches, and delete the ones you no longer need. You must have the command line version of Git installed for UGUI: Git Branch Deleter to work.

UGUI: Git Branch Deleter created by Jared Wilcurt.
Git was created by Linus Torvald and is maintained by the open source community.
